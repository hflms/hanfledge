## 2024-05-13 - Spinner Localization
**Learning:** Adding hardcoded Chinese strings to accessibility attributes (e.g., `aria-label="加载中"`) may create an inconsistent experience if the application relies on an English interface or dynamic localization, despite the user interface primarily using Chinese for its default text. It's safer to avoid hardcoded locale strings in global utility components unless verified against the specific language context they are rendered in.
**Action:** When adding `aria-label` to shared components like `LoadingSpinner`, prefer a language-agnostic approach or ensure it strictly aligns with the intended localization strategy of the specific page, even if the general memory dictates defaulting to Simplified Chinese.
