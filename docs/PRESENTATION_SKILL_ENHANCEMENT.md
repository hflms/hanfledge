# 演示文稿生成技能增强说明

## 功能概述

演示文稿生成技能现已支持**美观的 Reveal.js 渲染**,提供专业级的视觉呈现效果。

## 核心增强

### 1. Reveal.js 完整支持

#### 垂直幻灯片
使用 `--` 创建垂直幻灯片,用于展开细节:

```markdown
## 主题概述

核心内容

--

### 细节 1

详细说明

--

### 细节 2

更多细节
```

**效果:** 主题页面可以向下展开,查看更多细节

#### HTML 样式增强
支持内联 HTML 样式:

```markdown
<div style="text-align: left;">
- 左对齐内容
</div>

<div style="font-size: 1.2em; color: #2c3e50;">
强调文本
</div>
```

### 2. 美观设计系统

#### Emoji 图标系统

- 📚 📖 📝 - 学习、阅读、笔记

**状态标记:**
- ✅ ✓ ❌ - 正确、完成、错误
- 🔵 🔴 🟢 - 颜色标记
- ⚡ 🚀 💪 - 强调、快速、力量

**互动相关:**
- 🤔 💭 🗣️ - 思考、讨论、发言
- 📊 📈 📉 - 数据、增长、下降

**成长阶段:**

### 3. 完整示例

```markdown
<slides>
# 📚 光合作用

**生物学 · 第3章**

<small>张老师 | 2026-03-07</small>

> 备注：问候学生,介绍本节课的学习目标。

---

## 🎯 学习目标

<div style="text-align: left;">

- 🎓 **理解**：光合作用的基本概念
- 🔍 **掌握**：光合作用的化学方程式
- 💡 **应用**：解释植物生长现象

</div>

> 备注：逐条讲解学习目标,让学生明确核心任务。

---

## 核心概念：光合作用

<div style="text-align: left;">

**📖 定义：** 植物利用光能,将二氧化碳和水转化为葡萄糖和氧气的过程

**🔑 关键要点：**
- ✓ 场所：叶绿体
- ✓ 原料：CO₂ + H₂O
- ✓ 产物：C₆H₁₂O₆ + O₂

</div>

> 备注：用"植物的厨房"比喻帮助学生理解。

--

### 💡 化学方程式

<div style="text-align: center; font-size: 1.2em;">

6CO₂ + 6H₂O + 光能 → C₆H₁₂O₆ + 6O₂

</div>

<div style="text-align: left; margin-top: 2em;">

**能量转换：**
- 光能 → 化学能
- 储存在葡萄糖中

</div>

> 备注：强调这是能量转换过程,不是简单的化学反应。

---

## 📊 光合作用 vs 呼吸作用

| 维度 | 光合作用 | 呼吸作用 |
|:----:|:--------:|:--------:|
| 场所 | 🌿 叶绿体 | 🔬 线粒体 |
| 原料 | ✅ CO₂ + H₂O | ✅ C₆H₁₂O₆ + O₂ |
| 产物 | ⭐ 葡萄糖 + O₂ | ⚡ CO₂ + H₂O + 能量 |
| 能量 | 🔵 吸收光能 | 🔴 释放能量 |

> 备注：引导学生发现两者是相反的过程,构成生态循环。

---

## 🤔 思考与讨论

<div style="font-size: 1.2em; color: #2c3e50;">

**问题：** 为什么植物在白天和晚上的作用不同?

</div>

<div style="text-align: left; margin-top: 2em;">

*💭 提示：*
- 光合作用需要什么条件?
- 呼吸作用什么时候进行?

</div>

> 备注：给学生2-3分钟讨论,收集典型回答。

---

## 📝 本章小结

<div style="text-align: left;">

**核心知识点回顾：**
1. ✅ 光合作用的定义和场所
2. ✅ 化学方程式和能量转换
3. ✅ 与呼吸作用的对比

**🎯 下节预告：** 光合作用的影响因素

</div>

> 备注：强调重点,布置课后观察任务。

</slides>
```

## 使用方法

### 1. 创建学习活动

在教师后台创建学习活动时:
1. 选择章节和知识点
2. 选择技能: **演示文稿生成**
3. 发布活动

### 2. 学生学习

学生加入活动后:
1. AI 自动生成演示文稿
2. 点击 "📊 立即查看演示文稿"
3. 使用工具栏控制:
   - **⛶ 全屏** - 进入全屏播放
   - **收起** - 关闭演示文稿
   - **🔄 重新生成** - 请求重新生成

### 3. 导航控制

**键盘快捷键:**
- `→` / `Space` - 下一张 (水平)
- `←` - 上一张 (水平)
- `↓` - 下一张 (垂直)
- `↑` - 上一张 (垂直)
- `ESC` / `F` - 退出全屏
- `O` - 概览模式
- `?` - 显示帮助

- 统一的图标系统

## 故障排查

### 问题 1: 样式不生效

**原因:** HTML 标签未正确闭合

**解决:**
```markdown
<!-- 错误 -->
<div style="text-align: left;">
内容

<!-- 正确 -->
<div style="text-align: left;">

内容

</div>
```

### 问题 2: 垂直幻灯片不显示

**原因:** 分隔符使用错误

**解决:**
```markdown
<!-- 错误: 使用了 --- -->
## 主题
---
### 细节

<!-- 正确: 使用 -- -->
## 主题
--
### 细节
```

### 问题 3: 表格对齐问题

**原因:** 未使用居中对齐语法

**解决:**
```markdown
<!-- 错误 -->
| 列1 | 列2 |
|-----|-----|

<!-- 正确 -->
| 列1 | 列2 |
|:---:|:---:|
```
