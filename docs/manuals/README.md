# Hanfledge 角色操作手册集合

欢迎使用 Hanfledge AI-Native 教育平台文档库。本目录包含系统中各核心角色的详细操作手册：

## 📚 目录

- **[系统管理员 (SYS_ADMIN) 操作手册](./SYS_ADMIN_MANUAL.md)**
  面向负责全平台部署、多租户（学校）搭建与全局监控的最高权限用户。
- **[学校管理员 (SCHOOL_ADMIN) 操作手册](./SCHOOL_ADMIN_MANUAL.md)**
  面向负责单体学校内部组织架构（班级）、人员（师生账号）管理及全校宏观学情监控的用户。
- **[教师 (TEACHER) 操作手册](./TEACHER_MANUAL.md)**
  面向负责核心课程内容建设、苏格拉底式教学活动创编、技能与知识图谱维护、以及对学生进行个性化干预和学情分析的教育工作者。
- **[学生 (STUDENT) 操作手册](./STUDENT_MANUAL.md)**
  面向平台的主要受众，指导其如何参与智能对话活动、通过知识图谱查漏补缺、管理错题本及查看学习成就。

## 🔍 如何使用
请根据您在系统内部分配的权限角色，点击对应的文档链接查看完整指南。文档内容紧扣系统现有的 KA-RAG 管线、RBAC 权限划分以及业务模块（如知识树、成就、插件市场等）进行编写。
